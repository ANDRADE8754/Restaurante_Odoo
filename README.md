# Restaurante La Comanda (Grupo 3)
Sistema de gestión de restaurante, pedidos a domicilio y reservas desarrollado en Odoo 18.

## Requisitos Previos
- Docker Desktop
- Git

## Instalación
1. Clonar el repositorio: `git clone https://github.com/ANDRADE8754/Restaurante_Odoo.git`
2. Ingresar a la carpeta: `cd Restaurante_Odoo`
3. Levantar los servicios: `docker compose up -d`
4. Acceder en el navegador a `http://localhost`
